## How We Process Images

### Client-Side Processing
1. You select an image on your device
2. Your browser reads the image using JavaScript
3. Image processing happens in your browser's memory
4. Processed image is created in your browser
5. You download the result to your device

**At NO point does your image leave your device!**

### Technical Details
- We use HTML5 Canvas API for processing
- All computations run in your browser
- No server uploads or cloud processing
- Works offline after initial page load
