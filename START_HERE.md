## 🎨 Customization

### Change Colors
Edit `tailwind.config.js`:
```javascript
extend: {
  colors: {
    'your-color': '#hexcode',
  },
}
```

### Modify Animations
Edit `src/index.css`:
```css
@keyframes your-animation {
  /* your keyframes */
}
```

### Update Components
Edit `src/FormFit.jsx` - search for `className` to find styling
