## 🚀 How to Use the New Features

### Dark Mode
1. Click the sun/moon icon in the top-right corner
2. Theme changes instantly
3. Preference persists during session

### Aspect Ratio Lock
1. Upload an image
2. Check the "Lock aspect ratio" checkbox
3. Change width - height adjusts automatically
4. Change height - width adjusts automatically

### Quick Presets
1. Upload an image first
2. Click any preset button
3. Settings are applied automatically
4. Click "Process Image" to apply

### Format Selection
1. Choose your desired output format (JPEG/PNG/WebP)
2. Process your image
3. Download will use the selected format

### Quality Control
1. In Resize mode, use the quality slider
2. Drag left for smaller file size
3. Drag right for better quality
4. Sweet spot is usually 80-92%

### Keyboard Shortcuts
1. Copy an image (Ctrl+C)
2. In PixelPerfect, press Ctrl+V to paste
3. Adjust settings
4. Press Enter to process
5. Press Ctrl+S to download
