## 🚀 Basic Usage

### Uploading an Image

**Method 1: Drag & Drop**
1. Drag an image file from your computer
2. Drop it onto the upload area
3. Done! ✅

**Method 2: Click to Browse**
1. Click anywhere on the upload area
2. Select an image from your file browser
3. Click "Open"

**Method 3: Paste from Clipboard**
1. Copy an image (Ctrl+C)
2. Press Ctrl+V anywhere in PixelPerfect
3. Image is automatically loaded

---

### Resize Mode 📐

**Perfect for:** Changing image dimensions

1. Upload your image
2. Click "📐 Resize" button
3. Set your desired width and height
4. Use the quality slider (80-100% recommended)
5. Choose output format (JPEG/PNG/WebP)
6. Click "✨ Process Image"
7. Download your resized image

**Example Use Case:**
- Resize a 4000×3000 photo to 800×600 for email

---

### Compress Mode 🗜️

**Perfect for:** Reducing file size

1. Upload your image
2. Click "🗜️ Compress" button
3. Set maximum file size in KB (e.g., 50)
4. Choose output format
5. Click "✨ Process Image"
6. Watch the progress bar
7. Download your compressed image

**Example Use Case:**
- Compress a 2MB image to under 50KB for passport photo

## 🎨 Advanced Features

### Aspect Ratio Lock 🔒

**What it does:** Maintains image proportions to prevent distortion

**How to use:**
1. Check the "🔒 Lock aspect ratio" box
2. Change width → height adjusts automatically
3. Change height → width adjusts automatically

**Example:**
- Original: 1600×900 (ratio 1.78)
- Change width to 800 → height becomes 450
- Perfect proportions maintained! ✅

---

### Quality Slider 🎯

**What it does:** Controls output image quality

**Guide:**
- **100%** - Maximum quality, larger file size
- **80-92%** - Sweet spot (recommended)
- **50-79%** - Noticeable quality loss
- **1-49%** - Significant quality loss

**Best Practices:**
- Photos: 80-85%
- Graphics: 90-95%
- Screenshots: 85-90%
- Documents: 100%

---

## 💡 Tips & Tricks

### For Best Quality

1. **Start with high resolution**
   - Downscaling = good
   - Upscaling = bad (pixelated)

2. **Use appropriate formats**
   - Photos → JPEG
   - Transparency → PNG
   - Modern web → WebP

3. **Quality settings**
   - Start at 85% quality
   - Adjust based on results
   - Check file size vs quality

4. **Aspect ratio**
   - Always lock for resize
   - Prevents weird stretching
   - Maintains professional look

### For Smallest File Size

1. **Choose JPEG format**
   - Best compression for photos

2. **Lower quality setting**
   - Try 70-80% range
   - Check visual quality

3. **Resize first**
   - Smaller dimensions = smaller file
   - Then compress if needed

4. **Use WebP**
   - 25-35% smaller than JPEG
   - Same quality

## 🐛 Troubleshooting

### Image looks pixelated
- **Cause:** Trying to upscale (make larger)
- **Solution:** Start with higher resolution image

### File size too large
- **Resize Mode:** Lower quality slider
- **Compress Mode:** Set smaller KB limit
- **Format:** Try JPEG instead of PNG

### Image is distorted
- **Cause:** Aspect ratio lock is off
- **Solution:** Enable "🔒 Lock aspect ratio"

### Can't paste image
- **Cause:** Browser doesn't have clipboard access
- **Solution:** Use drag & drop or click to browse

### Dark mode not working
- **Solution:** Try refreshing the page
- Click the theme toggle button again
